Setting up your own GitHub page from scratch (or migrating from an existing website) is pretty easy!

Step 1

Sign in on GitHub

If you've already signed in, you can skip this step.

Step 2

Create an organization

You can give any name to your organization (but try to keep it nice for other people!)

Step 3

Create a new repository

Set the owner to your organization and the name to [orgname].github.io (replacing [orgname] with the name of the organization itself).

Step 4

Now that you've setup your repository, try pushing this zip's contents to it to test your setup!
